## ✨ Features

Wegent Browser Power seamlessly integrates **GitLab**, **Jira**, and **DingTalk** into your browser, breaking down barriers between tools and delivering AI capabilities right where you work.

Powered by Wegent AI's advanced large language model, this tool aims to empower the entire product development and research workflow.

## 📦 Installation

### Chrome Web Store (Recommended)

Go to [Chrome Web Store](https://chromewebstore.google.com/detail/wegent-browser-power/ambidpbogbcdlephgniokkakghccheoi) and click **"Add to Chrome"** button to install.

## 🔧 Post-Installation Setup

1. Click the extension icon in the Chrome toolbar to open the settings panel
2. Configure API:
   - **Auto Configuration**: Open Wegent in browser, click the extension icon > `Auto Configure` button (Recommended)
   - **Manual Configuration**: Enter Wegent API URL and API Key
3. (Optional) Enter the extension options page to customize AI actions for DingTalk, GitLab, and Jira
